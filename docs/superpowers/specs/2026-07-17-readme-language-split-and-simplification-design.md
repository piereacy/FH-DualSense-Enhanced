# README 语言拆分与整体精简设计

## 目标

将 GitHub 首页从三种语言拼接的长文档改为简洁的英文用户指南，并为简体中文、日语分别保留独立页面。README 只帮助普通用户判断项目用途、完成安装和游戏配置、解决最常见问题，不再承担功能清单、实现说明或开发手册的职责。

## 文件和语言入口

- 根目录 `README.md` 是默认英文首页。
- `docs/ReadmeZH.md` 是完整的简体中文页面。
- `docs/ReadmeJA.md` 是完整的日语页面。
- 三个页面顶部都提供 `English | 简体中文 | 日本語` 链接，并将当前语言显示为非链接强调文本。
- 删除重复的 `docs/ReadmeEN.md`。英文只有根 README 一个权威来源。

## 内容结构

三个页面使用相同顺序和相同信息层级：

1. 项目名称、图标和一句话用途。
2. 一段项目定位，说明它基于 `Forza-Horizon-DualSense-Python 1.6.2`，并参考 `HorizonHaptics 1.3.0`。
3. 最多六条核心亮点，只描述用户可感知的能力：自适应扳机、遥测握把触觉、路面与轮胎状态、方向碰撞、USB/Bluetooth、配置与内置更新。
4. Windows 推荐下载，链接到最新 Release，并使用稳定资产模式 `FH-DualSense-Enhanced-R<n>.exe`，避免 README 因每个版本改名。
5. 必需游戏设置：Steam Input、Forza Data Out 的 `127.0.0.1:5300`、启动顺序，以及游戏原生振动可能掩盖方向反馈的简短提醒。
6. USB 与 Bluetooth 的一句话差异。只说明两者均支持扳机和遥测触觉，不解释 PCM、HID report、队列、CRC 或量化算法。
7. 最多五项常见故障：无 UDP、端口占用、找不到手柄、USB 触觉音频错误、Bluetooth fallback。
8. 一段可选的 DualSense 按键图标 Mod 提醒。
9. 上游来源、HorizonHaptics 和 vDS 归属、第三方声明及自定义非商业许可证。

ZUV 与 Linux 只在下载部分各保留一条简短的备用入口，不展开环境补全、udev 命令或构建流程。需要维护者执行的安装、测试和构建命令继续由 `AGENTS.md` 与架构文档承担。

## 明确删除的内容

- 三种语言在同一文件中的锚点正文。
- “后台行为”及关闭游戏、最小化托盘等小设置说明。
- Default 自动保存、退出另存、恢复默认、首次语言检测、滚轮布局、更新白点等界面细节。
- R1 到 R4 的版本体系历史和逐版本功能说明。
- GT7 ABS wall、EWMA、红线优先级等算法级清单。
- `HapticPcmRenderer`、Bluetooth report `0x36`、398 字节布局、单槽队列和兼容回退实现过程。
- 完整握把分层原理、开发环境、测试、ZUV 构建和 Windows 打包命令。
- 大段防火墙教程、Linux udev 命令和低概率故障说明。

必要信息应直接写清楚，不使用折叠区把被删除的长内容藏起来。

## 篇幅与一致性约束

- 每种语言控制在约 80 到 120 行，英文正文目标不超过约 900 词。
- 三个页面保持同一组一级和二级主题，不要求逐句机械翻译，但功能事实、链接、端口和许可证必须一致。
- README 不硬编码尚未发布版本的具体 EXE 文件名，只使用 Release 链接和 `R<n>` 资产模式。
- 文档契约测试改为验证英文默认首页、三个独立语言入口、必要用户信息和被禁止的实现细节，不再要求长篇 R4 内部实现说明。

## GitHub 更新方式

GitHub 默认分支是 `main`，而当前 R4 工作位于 `feat/r4-ui-updater-haptics`。README 重构先在当前分支实现和验证，再以独立、可移植的文档提交同步到 `main`；不借此合并尚未发布的 R4 业务代码。更新 `main` 后运行适用于该分支的文档测试，再推送 `origin/main`，使仓库首页立即以英文显示。

## 验证

- 检查三个语言链接在 GitHub 相对路径下能相互到达。
- 检查根 README 不再包含中文或日语的整段副本。
- 检查 `docs/ReadmeEN.md` 已删除，`docs/ReadmeZH.md` 不再只是跳转占位页。
- 运行 README、打包、许可证相关定向测试及全量 `pytest`。
- 运行 `git diff --check`，检查没有新 em dash、断链或业务代码改动。
- 推送后通过 GitHub 默认分支确认英文首页和中日文链接。
